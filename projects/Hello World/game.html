About This Project

This is a demonstration of the GUTS (Gamedev Ultimate Toolkit System) ECS architecture and data-driven collections.

What you're seeing:

  • Collections defined in the GUTS editor are loaded and displayed
  • Data is stored as JSON and can be edited in the editor
  • The ECS (Entity-Component-System) architecture creates entities from collection data
  • Everything is data-driven and customizable

How to extend this:

1. Open the GUTS editor (index.html) 2. Select "Hello World" project 3. Add new collection types or objects 4. Refresh this page to see your changes!
